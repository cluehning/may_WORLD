# 🌍 Global Crisis & Conflict Map for MUN Delegates

An interactive dashboard that visualizes global crises and geopolitical risks using **official international datasets** (World Bank, UNHCR, IDMC, IMF).

Built with **Dash + Plotly**, this tool helps Model United Nations (MUN) delegates quickly identify global hotspots, understand key indicators, and support research, speeches, and resolution drafting.

---

# Features

## 🌐 Interactive World Map
- Visual world map showing crisis intensity by country  
- Click on countries to explore detailed data  
- Dynamic updates based on selected filters  

## 📊 Real Data Sources
- Uses **official, internationally recognized datasets only**
- No paid or restricted APIs  
- Fully reproducible and transparent  

### Included Indicators:
- War / armed conflict  
- Humanitarian pressure (displacement data)  
- Violence / instability risk  
- Refugee outflows  
- Trade / tariff pressure  

---

## Filtering System
- Select one or multiple crisis categories  
- Compare different types of global risk  
- Adjust severity threshold with slider  

---

## Analytical Tools
- Combined crisis score (normalized)  
- Top countries based on risk  
- Regional comparison  

---

## Country Intelligence Panel
Click any country to see:

- All selected indicators  
- Raw values (real data)  
- Interpretable scores
